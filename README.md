# Tienda de Videojuegos - Gestión de Inventario

Este proyecto es una aplicación de consola en Java que permite gestionar productos, clientes, proveedores, compras y ventas en una tienda de videojuegos utilizando una base de datos MySQL.

## Características
- CRUD de productos, clientes y proveedores
- Registro de compras y ventas
- Actualización automática de stock
- Interfaz de consola interactiva

## Requisitos
- Java 8 o superior
- MySQL Server
- Conector JDBC para MySQL


## Uso
Al iniciar, se mostrará un menú con opciones para gestionar productos, clientes, proveedores, compras y ventas. Puedes seleccionar una opción introduciendo el número correspondiente.

## Estructura del Proyecto
- `Main.java` - Punto de entrada del programa
- `Conexion.java` - Configuración de la conexión con MySQL
- `Producto.java` - Gestión de productos
- `Cliente.java` - Gestión de clientes
- `Proveedor.java` - Gestión de proveedores
- `Compra.java` - Gestión de compras
- `Venta.java` - Gestión de ventas
